# 손익분기점(BEP) 분석 상세 정리

## 1. 개요

이 문서는 기존 방식과 변경 방식을 비용 관점에서 비교하여,  
변경안 도입 시 필요한 초기 투자금이 언제 회수되는지,  
즉 손익분기점(BEP, Break-Even Point)이 언제인지 계산한 내용입니다.

또한 3년, 4년, 5년 시점에서 두 방식의 총비용을 비교하여  
어느 방식이 더 경제적인지 자세히 정리합니다.

---

## 2. 주어진 조건

### 2.1 기존 방식
- 3년 동안 발생하는 총비용: **94,140,000원**

### 2.2 변경 방식
- 3년 동안 발생하는 운영비: **10,800,000원**
- 도입 시 필요한 초기 투자비: **80,000,000원**

---

## 3. 용어 정리

## 3.1 기존 비용
현재 사용 중인 방식에서 드는 비용입니다.

- 여기서는 3년 총비용이 **94,140,000원**입니다.

## 3.2 변경 비용
새로운 방식으로 바꿨을 때 드는 비용입니다.

- 처음 한 번 드는 **초기 투자비**
- 사용하면서 계속 드는 **운영비**

## 3.3 초기 투자비
도입 초기에 한 번만 지급하는 금액입니다.

- 여기서는 **80,000,000원**

## 3.4 운영비
도입 후 유지와 사용을 위해 계속 발생하는 비용입니다.

- 여기서는 3년 동안 **10,800,000원**

## 3.5 연간 비용
총비용을 1년 기준으로 나눈 값입니다.

예:
```text
연간 비용 = 총비용 ÷ 기간
```

## 3.6 연간 절감액
기존 방식보다 변경 방식이 1년에 얼마나 비용을 줄여주는지를 뜻합니다.

예:
```text
연간 절감액 = 기존 연간비용 - 변경 연간운영비
```

## 3.7 손익분기점(BEP)
처음 투자한 돈을 매년 절약되는 금액으로 모두 회수하는 시점입니다.  
쉽게 말해 **본전 찾는 시점**입니다.

---

## 4. 계산에 사용하는 핵심 공식

## 4.1 기존 연간비용 공식

```text
기존 연간비용 = 기존 3년 총비용 ÷ 3
```

값 대입:

```text
기존 연간비용 = 94,140,000 ÷ 3 = 31,380,000원
```

즉, 기존 방식은 1년에 **31,380,000원**이 든다고 볼 수 있습니다.

---

## 4.2 변경 연간운영비 공식

```text
변경 연간운영비 = 변경 3년 운영비 ÷ 3
```

값 대입:

```text
변경 연간운영비 = 10,800,000 ÷ 3 = 3,600,000원
```

즉, 변경 방식은 1년에 **3,600,000원**의 운영비가 듭니다.

---

## 4.3 연간 절감액 공식

```text
연간 절감액 = 기존 연간비용 - 변경 연간운영비
```

값 대입:

```text
연간 절감액 = 31,380,000 - 3,600,000 = 27,780,000원
```

즉, 변경안을 도입하면 **1년에 27,780,000원씩 절감**됩니다.

---

## 4.4 손익분기점 공식

```text
손익분기점(년) = 초기 투자비 ÷ 연간 절감액
```

값 대입:

```text
손익분기점 = 80,000,000 ÷ 27,780,000 = 약 2.88년
```

즉, 약 **2.88년 후 손익분기점 도달**입니다.

개월 수로 바꾸면:

```text
2.88 × 12 = 34.56개월
```

즉, 약 **34.6개월**, 대략 **2년 10개월~11개월**입니다.

---

## 5. 총비용 계산 공식

어떤 시점의 총비용은 아래와 같이 계산합니다.

## 5.1 기존 방식 총비용

```text
기존 총비용 = 기존 연간비용 × 사용연수
```

## 5.2 변경 방식 총비용

```text
변경 총비용 = 초기 투자비 + (변경 연간운영비 × 사용연수)
```

## 5.3 차이 계산

```text
차이 = 기존 총비용 - 변경 총비용
```

- 차이가 플러스이면 변경안이 더 유리
- 차이가 마이너스이면 기존안이 더 유리
- 차이가 0이면 손익분기점 도달

---

## 6. 3년 시점 계산

## 6.1 기존 방식 총비용

```text
31,380,000 × 3 = 94,140,000원
```

## 6.2 변경 방식 총비용

```text
80,000,000 + (3,600,000 × 3)
= 80,000,000 + 10,800,000
= 90,800,000원
```

## 6.3 차이

```text
94,140,000 - 90,800,000 = 3,340,000원
```

## 6.4 해석
3년 시점에는 **변경 방식이 3,340,000원 더 유리**합니다.

---

## 7. 4년 시점 계산

## 7.1 기존 방식 총비용

```text
31,380,000 × 4 = 125,520,000원
```

## 7.2 변경 방식 총비용

```text
80,000,000 + (3,600,000 × 4)
= 80,000,000 + 14,400,000
= 94,400,000원
```

## 7.3 차이

```text
125,520,000 - 94,400,000 = 31,120,000원
```

## 7.4 해석
4년 시점에는 **변경 방식이 31,120,000원 더 유리**합니다.

---

## 8. 5년 시점 계산

## 8.1 기존 방식 총비용

```text
31,380,000 × 5 = 156,900,000원
```

## 8.2 변경 방식 총비용

```text
80,000,000 + (3,600,000 × 5)
= 80,000,000 + 18,000,000
= 98,000,000원
```

## 8.3 차이

```text
156,900,000 - 98,000,000 = 58,900,000원
```

## 8.4 해석
5년 시점에는 **변경 방식이 58,900,000원 더 유리**합니다.

---

### 9.1 연간 기준 요약

- 연간 절감액: **27,780,000원**

### 9.2 손익분기점

## 11. 실무용 공식 정리

### 손익분기점 계산식

```text
손익분기점 = 초기 투자비 ÷ (기존 연간비용 - 변경 연간운영비)
```

이번 사례에 대입하면:

```text
손익분기점 = 80,000,000 ÷ ((94,140,000 ÷ 3) - (10,800,000 ÷ 3))
```

## 12. 아주 쉽게 설명한 뜻

이 내용을 아주 쉽게 말하면 아래와 같습니다.

- 기존 방식은 매년 돈이 많이 드는 방식입니다.
- 변경 방식은 처음에 큰돈이 들어가지만, 이후에는 운영비가 적게 듭니다.
- 그래서 처음에는 변경안이 부담스러워 보일 수 있습니다.
- 하지만 매년 **27,780,000원**씩 절약되기 때문에
- 약 **2.88년**이 지나면 초기 투자비 **80,000,000원**을 모두 회수합니다.
- 그 이후부터는 변경안이 계속 더 이득입니다.

---

## 13. 결론

이번 조건에서는 변경안에 초기 투자비 **80,000,000원**이 필요하지만,  
연간 절감액이 **27,780,000원**으로 크기 때문에  
**약 2.88년(약 34.6개월)** 에 손익분기점에 도달합니다.

또한 3년, 4년, 5년 시점 비교 결과 모두 변경안이 유리하며,  
사용 기간이 길어질수록 절감 효과는 더 커집니다.

따라서 **중장기적으로는 변경안의 경제성이 높다**고 판단할 수 있습니다.
